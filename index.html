Getting Started

To run this Hugo project:

  1. Install Hugo: brew install hugo (macOS) or download from hugo.io
  2. Navigate to the project directory: cd norway-space-agency
  3. Start the development server: hugo server
  4. Open http://localhost:1313 in your browser
